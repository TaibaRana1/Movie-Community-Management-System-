# Movie Community Management System

The Movie Community Management System is a comprehensive platform that merges entertainment management with community engagement. It allows users to create profiles, manage personal watchlists, review and rate movies, participate in discussion forums, host or join watch parties, and engage in private messaging. Real-time notifications and recommendations enhance user experience, while history tracking supports analytics on viewing trends and trending content.

Admins have dedicated tools for content moderation, user management, report generation, and auditing actions to maintain platform integrity and transparency. The system features a robust MySQL database backend, a Node.js application layer, and a functional frontend built with HTML and CSS.

Key features include:

User profiles, friendships, and social activity tracking

Movie catalog with genres, watchlists, ratings, and reviews

Discussion forums, posts, comments, likes, and notifications

Private messaging between confirmed friends

Event hosting and participation for watch parties

Personalized movie recommendations and trending content

Admin dashboard with moderation tools, analytics, and audit trails

Technologies used: Node.js, MySQL, HTML, CSS, LucidChart/draw.io.
